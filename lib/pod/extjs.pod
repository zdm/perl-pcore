=head2 BufferedStore

=over

=item * In ExtJS v6.2.0 Do not use C<store.reload()> method, because it leads to issue two requests to the server (L<https://www.sencha.com/forum/showthread.php?469984-Buffered-store-ALWAYS-send-two-requests-on-load-event-if-all-data-fetched&p=1318406#post1318406>). =item * To load bufferedStore in the single request you need to adjust pageSize and leadingBufferZone.
